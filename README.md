# Color Flipper App

A simple, interactive web app that randomly changes the page background color with one click. Switch between **named CSS colors** 

## Live Demo
---

## Features

- **Instant color flip** on button click  
- Toggle between **Simple Colors** (`red`, `blue`, `lavender`) and **Hex Colors** (`#1a2b3c`)  
- Displays current color name or hex code  
- Smooth CSS transition animation  
- Fully responsive – works on mobile & desktop  
- No dependencies – pure HTML, CSS, and JavaScript

---

## Tech Stack

| Tech        | Description                     |
|-------------|---------------------------------|
| `HTML5`     | Page structure                  |
| `CSS3`      | Styling & animations            |
| `JS`        | Logic & DOM manipulation        |

---

## How to Use

1. Open the app in any modern browser.
2. Click the **"Flip Color"** button.
3. Watch the background change instantly!
4. The current color is shown below the button.
